## ⚙️ 기능
### 💡 원격 제어 스위치
- n개의 스위치를 앱으로 원격 제어
### 🏠 IoT 스마트홈
- 스마트 도어락
- 스마트 온습도계
- LED 벽시계
- 조명 제어

  - 스마트 전등
